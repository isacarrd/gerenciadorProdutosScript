# 📦 Gerenciador de Produtos - Fase 1 (React + Mock)

Bem-vindo à **Fase 1** da evolução do nosso Gerenciador de Produtos. 

## 🚀 O que está sendo feito nesta fase?
Aqui, abandonamos o HTML/JS puro e introduzimos o **ReactJS com Tailwind CSS**. O foco desta etapa é estritamente **Lógica de Interface e UI**. 

Não temos banco de dados ainda. Utilizamos um arquivo JavaScript estático (`mock`) contendo dados inalteráveis. O objetivo é garantir que a aplicação seja 100% visualizável, testando as renderizações condicionais (como a mudança de cor quando o estoque é zero) e os filtros nativos do React.

## ⚙️ Tecnologias Utilizadas
- ReactJS (Componentização e Estados)
- Tailwind CSS (Estilização baseada em utilitários)
- Mock de Dados Estáticos (JSON/JS)
